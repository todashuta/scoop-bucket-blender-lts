README
======

Usage
-----

__Add Bucket__

```
$ scoop bucket add blender-lts https://github.com/todashuta/scoop-bucket-blender-lts.git
$ scoop update
```

__Install Blender 4.5 LTS__

```
$ scoop install blender4.5
```

__Install Blender 4.2 LTS__

```
$ scoop install blender4.2
```
